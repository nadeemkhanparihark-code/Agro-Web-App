# Agrofarm - Sustainable Agriculture Website

A modern React + Vite agriculture website promoting sustainable farming practices.

## Features

- Responsive design (320px - 1440px+)
- React Router for navigation
- Custom CSS with CSS variables
- Hamburger menu for mobile
- Data-driven components using JSON

## Getting Started

```bash
npm install
npm run dev
```

## Pages

- Home
- About
- Services
- Products
- Gallery
- Blog
- Contact

## Tech Stack

- React 19
- Vite
- React Router DOM
- Custom CSS (no Tailwind)
